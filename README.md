# Static EgyLifts-like UI (HTML/CSS/JS)

واجهة Static شبيهة بشكل `egylifts.com` باستخدام **HTML + CSS + JavaScript فقط** (بدون Back-end).

## التشغيل

### الطريقة 1 (أسهل)
- افتح ملف `index.html` مباشرة في المتصفح.

### الطريقة 2 (سيرفر محلي)
لو عندك Node.js:

```bash
npx serve .
```

- رقم واتساب مضبوط في `app.js` داخل المتغير: `WHATSAPP_PHONE_E164`
